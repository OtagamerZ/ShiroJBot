# Uptime

{% hint style="success" %}
### Nível de permissão: `Usuário`
{% endhint %}

### Permissões necessárias

* Básicas

### Aliases

* uptime

### Descrição

Serve para consultar quanto tempo faz desde a última reinicialização da Shiro.

### Exemplos

* `s!uptime`
  * Mostrará quanto tempo faz desde que a Shiro foi reiniciada.
